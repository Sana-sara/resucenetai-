# RescueNet AI+

RescueNet AI+ is a full-stack starter project with:
- **Frontend:** React + Vite
- **Backend:** Flask

## Structure

```text
frontend/
backend/
```

## Quick start

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```
